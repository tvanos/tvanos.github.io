Fanshawe College

The purpose of the document is to provide an outline of the critical systems and services that has designed for necessary functionality ranging from networking operation, domain administration, corporate security and telephony operations.

Highlights include:

  • Enterprise Virtualization
  • Network Security
  • Storage Management
  • Server Infrastructure Security
  • Virtualization
  • Active Directory Architecture
  • Corporate WAN Infrastructure
